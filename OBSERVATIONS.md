# Observations

What we're seeing as we test time awareness in real sessions.

## Time awareness ≠ time-shaped behavior

Claude can see the clock and announce it. But seeing time doesn't automatically change how it works. It still explores broadly, drafts multiple options, asks open questions with minutes left. The data alone isn't enough.

To address this, added a CLAUDE.md guideline that frames time data as context that should shape how Claude works, not just a number to announce.
